# heatgun

cheap heatgun, using soldering iron heater and 12v fan 

### Install
- 3dom download @babakhani/heatgun
- cd heatgun
- 3dom install 

### Settings: 
- printer: custom 
- supports: false
- resolution: 0.2mm
- infill: 20%
- material: abs
- filamentweight: 12g
- filamentsize: 1.75mm
